# space-fighters
Quick paced Space-Shooting Game (for 2 to 4 Players)

Controls:
P1: arrow keys (left, up, right)
P2: A, W, D
P3: J, I, L
P4: F, T, H

To play the game, you need Python3 and Pygame installed.
